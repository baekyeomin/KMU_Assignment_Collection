# [과제 B1] 한국어 형태소 분석기 개발

### 과제 개요
입력 텍스트 파일에서 한국어 조사와 어미를 분리해주는 형태소 분석기 개발 <br>
같은 경로에 미리 정의되어있는 조사(josa96.txt), 어미(eomi152.txt)를 불러온 뒤,   <br>
각 단어가 어떤 조사/어미로 끝나는지 찾아서 어근 + 접미사/조사 또는<br>
어근 + 접미사/어미 형태로 출력하도록 하는 코드를 python, C++로 구현함.  <br> 

### 예시 코드 (JosaEomi.py)와 다른점 
- 기존 josaEomi.py 코드에서 느낌표(!) 가 제거되지 않아  
  훔쳤다!, 적이다! 등의 문장이 정상 분리되지 않던 문제를 해결했습니다.  
- 문장부호 제거 대상에 !, ?, : 등을 추가하여 전처리 정확도를 개선했습니다.  
- 조사/어미 분리가 실패할 경우 , "==> 조사/어미 분리되지 않음" 메시지를 출력하도록 수정했습니다.
